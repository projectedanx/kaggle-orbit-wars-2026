Game Mechanics Planets produce ships each turn (proportional to their radius) Inner planets rotate around the central sun; outer planets are static Fleets fly in straight lines at a given angle from their source planet Fleet speed scales with fleet size (1 ship = 1/turn, larger fleets up to 6/turn) Combat: arriving fleet ships are subtracted from the planet's garrison. The Four Bugs to Fix First The tutorial names them explicitly. Here's the precise fix for each:
Bug 1 — No travel time compensation (the orbit intercept problem)
The sniper aims at nearest.x, nearest.y — the planet's current position. For orbiting planets, this is wrong. The fix requires predicting where the planet will be when your fleet arrives. Fleet travel time depends on fleet size (the nonlinear speed formula), so you need to iterate:
python import math
SUN = (50.0, 50.0) MAX_SPEED = 6.0
def fleet_speed(ships): if ships <= 1: return 1.0 return 1.0 + (MAX_SPEED - 1.0) * (math.log(ships) / math.log(1000)) ** 1.5
def predict_planet_position(planet, initial_planets, angular_velocity, turns): """Returns (x, y) of an orbiting planet after turns turns.""" # Find initial angle from center ip = next((p for p in initial_planets if p.id == planet.id), None) if ip is None: return planet.x, planet.y cx, cy = 50.0, 50.0 initial_angle = math.atan2(ip.y - cy, ip.x - cx) orbital_radius = math.sqrt((ip.x - cx)**2 + (ip.y - cy)**2) # Check if this planet orbits (orbital_radius + planet_radius < 50) if orbital_radius + planet.radius >= 50.0: return planet.x, planet.y  # Static planet new_angle = initial_angle + angular_velocity * turns return cx + orbital_radius * math.cos(new_angle), cy + orbital_radius * math.sin(new_angle)
def intercept_angle(source, target_planet, initial_planets, angular_velocity, ships): """Iteratively solve for the intercept angle accounting for orbit.""" speed = fleet_speed(ships) # Initial estimate: aim at current position tx, ty = target_planet.x, target_planet.y for _ in range(10):  # iterate to convergence dist = math.sqrt((source.x - tx)**2 + (source.y - ty)**2) travel_turns = dist / speed tx, ty = predict_planet_position( target_planet, initial_planets, angular_velocity, int(travel_turns) ) return math.atan2(ty - source.y, tx - source.x), (tx, ty) Bug 2 — Multiple planets targeting the same enemy (fleet duplication)
