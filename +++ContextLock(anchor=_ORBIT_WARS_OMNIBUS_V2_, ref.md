## Phase III: Phylogenetic Swarm \& KR-UCT Integration

### 3.1 KR-UCT Architecture for Orbit Wars

KR-UCT (Kernel Regression UCT), formalized by Yee, Lisý and Bowling at IJCAI 2016, is the structurally correct algorithm for this domain because it was designed for exactly the convergent conditions present in Orbit Wars: continuous action spaces, execution uncertainty, and the critical insight that **samples of execution uncertainty from one action provide information about any other action that could have generated that execution outcome** [web:26].

The core algorithmic structure maintains, for each node in the search tree, a dataset of evaluated actions $\{(a_i, v_i)\}$ where $a_i \in \mathcal{A}_{valid}$ is a valid $(m, \alpha)$ pair and $v_i$ is the simulated rollout return. The kernel regression estimate of action value for any query point $a$ is:

$$
\hat{E}[v | a] = \frac{\sum_{i} K(a, a_i) \cdot \bar{v}_{a_i} \cdot n_{a_i}}{\sum_{i} K(a, a_i) \cdot n_{a_i}}
$$

And the kernel density (effective sample coverage):

$$
W(a) = \sum_{i} K(a, a_i) \cdot n_{a_i}
$$

The UCB selection formula becomes:

$$
\text{action}^* = \arg\max_{a \in \mathcal{A}_{valid}} \hat{E}[v|a] + C\sqrt{\frac{\log \sum_b W(b)}{W(a)}}
$$

For Orbit Wars, the kernel function $K(a, a')$ must encode the **execution uncertainty model** of the orbital kinematic system. The natural choice is an anisotropic Gaussian RBF:

$$
K\left((m, \alpha), (m', \alpha')\right) = \exp\left(-\frac{(m - m')^2}{2\sigma_m^2} - \frac{(\alpha - \alpha')^2}{2\sigma_\alpha^2}\right)
$$

Where the bandwidths $\sigma_m$ and $\sigma_\alpha$ are calibrated to the sensitivity of the intercept function — specifically, to the velocity derivative magnitude $\left|\frac{\partial v}{\partial m}\right|$ evaluated at the current candidate mass. High derivative magnitude (rapid velocity change with mass) demands narrow $\sigma_m$ to avoid information bleeding across kinematically distinct regimes. This is the **Radial Basis Function Density Harvesting** mechanism: the kernel width is not a fixed hyperparameter but a dynamic, state-conditioned function of the local manifold curvature.

Non-zero values confirm the knot; zero values (a falsification condition) would indicate a separable action space safe for independent discretization.
**Resolution:** Joint $(m, \alpha)$ sampling with the 3-iteration implicit intercept solver, treating both as co-varying dimensions in the KR-UCT candidate set.

### Pattern 3 — Kernel Density Failure Propagation (KR-UCT Execution Uncertainty)

**Type:** Epistemic
**Mechanism:** API load variance and solar occlusion are structurally isomorphic as stochastic execution uncertainty distributions: both produce outcome distributions over the intended action that are **non-Gaussian and heavy-tailed** (solar annihilation is a hard zero; API timeout is a Dirac delta at zero return). The KR-UCT kernel $K(a, a')$ treats these as equivalent noise sources — failure outcomes sampled for one action $(m, \alpha)$ update the kernel density estimates of neighboring actions in $\mathcal{A}$ according to their RBF proximity, **propagating failure information across the continuous action space** rather than isolating it to a single evaluated point [web:26].
**Diagnostic:** Rate of successful solar-grazing paths (trajectories passing within $1.5 r_\odot$ of the Sun without annihilation) as a function of KR-UCT iteration count. Successful agents should show increasing solar-grazing success as kernel density builds around valid near-boundary actions.
**Resolution:** Calibrate $\sigma_\alpha$ to $\approx 0.05$ radians for near-boundary regions, narrowing the kernel to avoid blending valid and invalid boundary actions.

### Pattern 4 — Travel Time Blindness (Static Target Delusion)

**Type:** Kinematic-Epistemic
**Mechanism:** Agents that compute the target planet's current angular position $\theta_{planet}(t_0)$ and dispatch toward it commit Travel Time Blindness — they are solving the zeroth-order approximation of a problem that requires first-order (or higher) iterative convergence. The planet's angular velocity $\omega_{planet}$ and the fleet's mass-dependent travel time $t^*(m)$ jointly determine the true intercept position $\mathbf{q}^*(m, \alpha)$, which is a function of the decision variables themselves. Baseline agents ignoring this suffer a **systematic intercept offset** of:

$$
\Delta\theta_{miss} \approx \omega_{planet} \cdot t^*(m) \approx \omega_{planet} \cdot \frac{d}{v(m)}
$$

For a slow, heavy fleet ($m = 1$, $v = 1$) on a 50-unit transit, with $\omega_{planet} = 0.02$ rad/tick, the angular miss is $\Delta\theta_{miss} \approx 1.0$ radians — a full planet-diameter miss at most orbital radii.
**Diagnostic:** Compute empirical fleet interception rate as a function of fleet mass. Agents with Travel Time Blindness show declining interception rate at low mass (slow fleets miss further).
**Resolution:** 3-iteration Newton-Raphson solver on the implicit equation $t^* = d(\mathbf{p}_0, \mathbf{q}(t^*)) / v(m)$, with PDOG validity check applied at each intermediate iterate.

### Pattern 5 — Semantic Saponification via Linguistic Overshadowing

**Type:** Epistemic-Representational
**Mechanism:** Semantic Saponification describes the dissolution of non-convex geometric structure when the agent's decision logic is expressed in human-readable heuristics. Examples include: "attack the nearest planet," "send max fleet to the front," "hold back reserves." These statements have zero formal relationship to the actual kinematic manifold and introduce **Linguistic Overshadowing** — the human-language framing of the problem suppresses the mathematical structure that determines correct action selection. This is not a behavioral failure; it is a **representational failure** occurring at the level of the agent's cognitive encoding of the problem.
**Diagnostic:** Cross-entropy between the agent's action distribution and the KR-UCT kernel density estimate. High cross-entropy indicates the agent is sampling from a human-language-derived heuristic distribution rather than the geometrically valid action manifold.
**Resolution:** Enforce all decision logic through formal parameterization — every action is expressed as a tuple $(m, \alpha, t_{dispatch})$ subject to PDOG validation, with no natural-language heuristic layer in the execution path.
